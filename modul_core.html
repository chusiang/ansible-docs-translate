核心模組¶

這些模組是 ansible 團隊維護的核心模組，同樣也是 ansible 自帶的模組，在收到的的請求中，它們有比 “extras” 源更高的優先順序

核心模組的源碼託管在 Github 的 ansible-modules-core repo.

如果你確信你在核心模組上發現了一個 bug ，同時你使用的是最新的穩定版或者開發版本的 Ansible ，首先你需要看看 github.com/ansible/ansible-modules-core 上的 “issue tracker” ，確保你的 bug 還沒有被提交。如果還沒被提交， 非常感謝你的提交

你肯能更想去問問題，而不是提交 bug ，歡迎在 Ansible-project 的 google-group 裡諮詢 https://groups.google.com/forum/#!forum/ansible-project ，在 Ansible 的 #ansible_chanel 諮詢也是可以的，它們位於 irc.freenode.net 。開發方向的應該用類似的討論組，位於 <https://groups.google.com/forum/#!forum/ansible-devel>`_

這些模組的文件更新可以直接編輯在模組自身裡面，然後提交一個 pull 請求到源碼，不過你需要找到源碼樹的 “DOCUMENTATION” 塊
