## zh-cn

1. Build.

1. Inatall the [OpenCC](https://github.com/BYVoid/OpenCC).

        $ sudo apt-get install opencc
        
        # Mac OS X
        $ brew install opencc

2. Build.

        $ make zhtw_webdocs
        
3. Open the `htmlout-zhtw/index.html`.
